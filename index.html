SimpleExcel PHP

PHP library with simplistic approach for parsing/writing data from/to Microsoft Excel XML/CSV format

Download

You can download this project in either ZIP or TAR formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/faisalman/simple-excel-php

Authors

  • Faisalman <fyzlman@gmail.com>

Version

v0.2.2

Features

  • Parsing Microsoft Excel XML/CSV cells to PHP arrays
  • Writing Microsoft Excel XML/CSV cells from PHP arrays

Dependencies

  • PHP5++

Install

Just include this library on your PHP project with either include() or require()

Usage Example

Parsing an XML file, simplified: 

<?php

require_once('../src/SimpleExcel/SimpleExcel.php'); // load the main class file

$excel = new SimpleExcel('xml');                    // instantiate new object (will automatically construct the parser)

$excel->parser->loadFile('example.xml');            // load an XML file from server to be parsed

$foo = $excel->parser->getField();                  // get complete array of the table
$bar = $excel->parser->getRow(3);                   // get specific array from the specified row (3rd row)
$baz = $excel->parser->getColumn(4);                // get specific array from the specified column (4th row)
$qux = $excel->parser->getCell(2,1);                // get specific data from the specified cell (2nd row in 1st column)

?>
Writing an XML file, simplified: 

<?php

/**
 * Warning: note that there must be no data sent to browser
 * (be it an echo, HTML element, or even a whitespace)
 * before the writer->saveFile() method get called
 */
require_once('../src/SimpleExcel/SimpleExcel.php'); // load the main class file

$excel = new SimpleExcel('xml');                    // instantiate new object (will automatically construct the writer)

$excel->writer->setData(
    array
    (
        array('ID',  'Nama Kota',    'ID Wilayah'),
        array('1',   'Kab Bogor',    '1'         ),
        array('2',   'Kab Sukabumi', '1'         )
    )
);                                                  // add some data to the writer
$excel->writer->saveFile('example');                // force browser to save the file with specified name (example.xml)

?>

API

Constructor

  • constructParser($filetype)

    Construct a SimpleExcel Parser

  • constructWriter($filetype)

    Construct a SimpleExcel Writer

Parser

  • parser->getCell($row_num, $col_num)

    Get value of the specified cell. Returns an array or FALSE if cell doesn't exist

  • parser->getCellDatatype($row_num, $col_num)

    Get datatype of the specified cell. Returns an array or FALSE if cell doesn't exist

  • parser->getColumn($col_num, $val_only)

    Get data of the specified column as an array. Returns an array or FALSE if table doesn't exist

  • parser->getField()

    Get data of all cells as an array. Returns an array or FALSE if table doesn't exist

  • parser->getRow($row_num, $val_only)

    Get data of the specified row as an array. Returns an array or FALSE if cell doesn't exist

  • parser->loadFile($file_path)

    Load the file to be parsed. Returns TRUE if file exist and valid, FALSE if does'nt

Writer

  • writer->addRow($values)

    Adding one row of data

  • writer->saveFile($filename)

    Export the document

  • writer->setData($values)

    Set the table data

  • writer->setDocProp($prop, $val)

    [XML only] Set the document property
